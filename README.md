# Afri Smart Pay 💳
A digital payment gateway for the Afri Smart ecosystem.

## Overview
Afri Smart Pay enables users to:
- Send and receive payments
- Store digital balances
- Connect seamlessly with Afri Smart Market and other services

## Tech Stack
- Node.js (Express)
- MongoDB / Firebase (for data)
- Render (Hosting)
- WhatsApp / Afri Smart Connect Integration
